Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add adss modifications from the working branch to the staging area.
  • git pull recieves a branch's modifications into the local environment.
  • The origin keyword indicates the source of the pull is from our Github repo
  • git push moves all changes from the local branch onto our remote github branch.
